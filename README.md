# Clientes y Pedidos - SQL

## Descripción

Este proyecto consiste en la creación y gestión de una base de datos relacional para administrar clientes y sus pedidos utilizando MySQL.

El objetivo principal es practicar el uso de:

* Practicar queries con JOINS.
* Creación de bases de datos y tablas.
* Relaciones entre tablas mediante claves foráneas (Foreign Keys).
* Consultas con JOIN.
* Funciones de agregación (COUNT y SUM).
* Eliminación de registros relacionados.
* Ordenamiento y limitación de resultados.

---

## Tecnologías Utilizadas

* MySQL
* MySQL Workbench
* SQL

---

## Aprendizajes Obtenidos

Durante el desarrollo de este proyecto se reforzaron conocimientos sobre:

* Diseño de bases de datos relacionales.
* Claves primarias y foráneas.
* Relaciones uno a muchos.
* Consultas JOIN.
* Funciones de agregación.
* Manipulación de datos mediante INSERT, SELECT y DELETE.
* Uso de MySQL Workbench para modelado y generación automática de scripts mediante Forward Engineer.
